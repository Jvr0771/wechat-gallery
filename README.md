#WeChatGallery 简介
Android 自定义相册，实现拍照、图片选择（多选/单选）， 裁剪。主题色可配置。交互仿照微信 Android 版。

#感谢 
* 部分源码来自 [GalleyFinal 库](https://github.com/pengjianbo/GalleryFinal)；
